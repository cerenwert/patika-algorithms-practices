# Patika Veri Yapıları ve Algoritmalar

Bu repository, Patika.dev platformunda yer alan
**Veri Yapıları ve Algoritmalar** dersleri kapsamında
yaptığım alıştırmaları ve örnekleri içermektedir.

## İçerik
- Diziler (Arrays)
- Bağlı listeler (Linked List)
- Stack / Queue
- Recursive fonksiyonlar
- Temel sıralama algoritmaları

Amaç; algoritma mantığını pekiştirmek ve
GitHub üzerinde düzenli bir çalışma arşivi oluşturmaktır.
